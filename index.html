Pertanyaan UMUM

Jurusan Desain Komunikasi Visual (DKV) adalah program studi yang fokus pada penggabungan seni dan teknologi untuk menyampaikan pesan atau informasi secara visual. Mahasiswa DKV akan mempelajari berbagai aspek desain, seperti ilustrasi, tipografi, fotografi, animasi, dan multimedia, serta bagaimana mengaplikasikan elemen-elemen ini untuk media cetak dan digital.

Lulusan DKV memiliki beragam peluang karier di berbagai industri kreatif. Beberapa di antaranya adalah: Desainer Grafis, Art ,Director, Ilustrator,Desainer Web dan UI/UX dsb

Untuk sukses di jurusan DKV, mahasiswa perlu mengembangkan beberapa keterampilan penting, sperti teknik desain, komunikasi visual, manajeman waktu
